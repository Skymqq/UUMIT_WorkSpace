# 微服务系统架构 Excalidraw 交互模板使用指南

## 概述

本模板提供了一套完整的微服务系统架构可视化方案，使用 Excalidraw 手绘风格绘制，包含用户层、网关层、服务层和数据层四层架构，共 13 个节点和 6 条调用关系箭头。

## 模板结构

### 四层架构

1. **用户层** — Web浏览器、移动App、外部API调用方
2. **网关层** — API网关、负载均衡、认证中心
3. **服务层** — 用户服务、订单服务、支付服务、通知服务
4. **数据层** — MySQL主库、Redis缓存、消息队列、对象存储

### 调用关系

- **蓝色箭头** (#4dabf7) — API 调用（Web→网关、外部API→认证）
- **绿色箭头** (#51cf66) — 内部 RPC（网关→订单、订单→通知）
- **橙色箭头** (#ff922b) — 异步消息（支付→消息队列）

## 使用方式

1. 打开 [Excalidraw.com](https://excalidraw.com)
2. 点击左上角菜单 → 加载（Load）
3. 选择本模板文件即可开始编辑

- 微服务架构设计评审
- 系统重构方案讨论
- 技术方案文档插图
