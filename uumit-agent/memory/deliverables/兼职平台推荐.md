# 推荐 5 个适合大学生做的线上兼职平台

## 1. 猪八戒网（zbj.com）
- **核心特点**：国内最大的综合性技能众包平台，任务类型涵盖设计、开发、文案、翻译等。大学生可通过"八戒严选"或"八戒工场"接单，平台提供担保交易。
- **适合人群**：有设计/编程/写作等一技之长的在校生。
- **收入范围**：入门任务 20-100 元/单，熟练后可接千元级项目，月收入 500-5000 元。
- **靠谱指数**：★★★★☆（成立18年，平台担保交易，稳妥）

## 2. 腾讯搜活帮（souhuobang.qq.com）
- **核心特点**：腾讯官方旗下的微任务平台，主要做数据标注、文本校对、图像分类、语音转写等 AI 训练数据任务。无需专业技能，有手机就能做。
- **适合人群**：零基础、想利用碎片时间赚零花钱的大学生。
- **收入范围**：按量计费，熟练后 10-30 元/小时，月收入 300-2000 元。
- **靠谱指数**：★★★★★（腾讯官方平台，无押金，结算准时）

## 3. 百度众测（zhongce.baidu.com）
- **核心特点**：百度官方测试平台，主要做产品体验、功能测试、问卷调研、AI 效果评估等任务。任务简单，操作门槛低。
- **适合人群**：想轻松赚零花钱、愿意花时间做任务的学生。
- **收入范围**：任务单价 1-20 元不等，月收入 200-1500 元。
- **靠谱指数**：★★★★★（百度旗下，无需任何费用）

## 4. 阿里众包（ CrowdTest ）
- **核心特点**：阿里巴巴旗下众测平台，主要参与淘宝/天猫/支付宝等产品的内测任务，包括功能测试、体验反馈、bug 提交等。部分任务可远程、灵活安排。
- **适合人群**：对互联网产品感兴趣、细致认真的学生。
- **收入范围**：按任务结算，月收入 300-3000 元，优秀测试员收入更高。
- **靠谱指数**：★★★★☆（阿里体系，信誉好，但需要一定耐心）

## 5. 问卷星/腾讯问卷 付费调研
- **核心特点**：参与企业和研究机构的市场调研问卷，完成即可获得现金奖励。渠道包括腾讯问卷（wj.qq.com 的"赚钱"入口）、K任务、爱调研等。
- **适合人群**：愿意花时间填写问卷、表达真实意见的学生。
- **收入范围**：每份问卷 1-20 元，月收入 100-800 元。
- **靠谱指数**：★★★☆☆（部分渠道可靠，注意甄别，优先选择腾讯问卷等大平台）

---

## 附：防坑提醒
1. 任何要求**先交押金/培训费**的兼职都是诈骗
2. 优先选择**大平台官方渠道**，远离微信群、QQ群发布的来历不明任务
3. 注意保护个人信息，不要提供身份证照片、银行卡密码等敏感信息
4. 刷单类兼职违法且风险极高，切勿参与
5. 建议从 1-2 个平台开始尝试，稳定后再扩展
