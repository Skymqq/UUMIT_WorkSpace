# AI 编程助手实战手册：Cursor + Claude + Copilot 效率提升指南

## 第1章：AI 编程工具全景概览
### 1.1 主流 AI 编程助手对比
- Cursor：基于 VS Code 的 AI-first IDE，深度上下文理解
- GitHub Copilot：微软生态深度集成，代码补全最强
- Claude Code：Claude 驱动的终端编程助手
- 各有优劣，组合使用效果最佳

### 1.2 AI 编程的四个层次
- L1：代码补全（Copilot 自动补全）
- L2：代码生成（自然语言描述生成函数）
- L3：代码理解与重构（分析现有代码改进）
- L4：全栈开发（从需求到部署）

## 第2章：Cursor 高效编程实战
### 2.1 Cursor 核心功能
- Ctrl+K：在选中的代码上输入指令修改
- Ctrl+L：对话模式，选中代码作为上下文
- @ 引用：@Files @Folders @Docs @Web
- Composer 模式：同时编辑多个文件

### 2.2 实战技巧
- 写代码前先用注释描述逻辑
- 利用 @Web 获取最新 API 文档
- 用 Composer 做跨文件重构
- 设置 .cursorrules 定制 AI 行为

## 第3章：Prompt 工程进阶
### 3.1 编程专用 Prompt 模板
- 需求描述模板：角色 + 任务 + 约束 + 输出格式
- Bug 修复模板：现象 + 期望 + 代码 + 错误日志
- 代码审查模板：关注点 + 代码片段 + 质量标准
- 重构模板：原因 + 范围 + 目标格式

### 3.2 上下文管理
- 不要一次性给太多代码
- 分步骤引导 AI 理解业务逻辑
- 用注释标记关键决策点
